# 書籍検索アプリ
- 制作期間: 約２週間
- URL： https://fumiaki49.github.io/book-search/dist/index.html

## 概要
- 楽天ブックス書籍検索APIを使用して作成した書籍検索アプリです。
これは、株式会社Gizumoの研修で作成したアプリをもう一度復習とwebpackやgulpへの理解を深める目的でもう一度作りました。

## 機能
- 書籍検索機能
- 非同期通信機能
- ヘルプボタン（モーダル）
- show more button（次の20件の商品を取得）

## 使用した技術
- 楽天ブックス書籍検索API
- js / jQuery
- gulp-sass
- webpack
- babel-loader
